Compiling the Trusted Execution environment is combining several
key projects that are worth looking at in more detail:

- `EDK II Project`_ (UEFI)
- `Trusted Firmware-A`_
- `OP-TEE Trusted OS`_
- `Linux Kernel`_
- `Buildroot`_ Embedded Linux
